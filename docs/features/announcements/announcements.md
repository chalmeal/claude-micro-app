## 概要

管理者がシステム利用者へのお知らせを配信・管理する機能。お知らせにはカテゴリ（重要・情報・メンテナンス）があり、認証済みユーザー全員が閲覧できる。作成・編集・削除は管理者のみ。

---

## ユーザーストーリー

```
ユーザーとして、
システムに関する最新情報を把握するために、
お知らせ一覧を確認したい。

管理者として、
利用者に重要な情報を周知するために、
カテゴリを指定してお知らせを作成・編集・削除したい。
```

---

## 機能要件

### 必須（Must）

- [x] お知らせ一覧を表示できる（認証済みユーザー全員）
- [x] お知らせを作成できる（管理者のみ）
- [x] お知らせを編集できる（管理者のみ）
- [x] お知らせを削除できる（管理者のみ）
- [x] カテゴリ（important / info / maintenance）を指定できる
- [x] 管理画面からお知らせを管理できる（`/admin/announcements`）

### 対象外（Won't）

- お知らせの予約投稿
- 既読管理・通知機能

---

## 受け入れ条件

- [x] 未認証ユーザーがアクセスすると401エラー
- [x] 管理者以外がPOST/PUT/DELETEを実行すると403エラー
- [x] カテゴリは `important` / `info` / `maintenance` のみ受け付ける
- [x] 日付は `YYYY-MM-DD` 形式で指定する
- [x] 一覧は30件ずつページネーション表示

---

## 画面・遷移の概要

- 一般ユーザー向け一覧画面（`/announcements`）: 閲覧のみ
- 管理者向け一覧画面（`/admin/announcements`）: 作成・編集・削除操作
- 新規作成画面（`/admin/announcements/new`）
- 編集画面（`/admin/announcements/:id/edit`）
