## 概要

管理者が組織内のユーザーアカウントを管理する機能。ユーザーの一覧・詳細確認・新規作成・ロール/ステータス変更ができる。CSV一括インポートにも対応。

---

## ユーザーストーリー

```
管理者として、
組織内の全ユーザーを把握するために、
ユーザー一覧を確認・検索したい。

管理者として、
新規メンバーをシステムに招待するために、
ユーザーを作成して初期パスワード設定メールを自動送信したい。

管理者として、
退職者や権限変更に対応するために、
ユーザーのロール・ステータスを変更したい。

管理者として、
大量のユーザーを効率よく登録するために、
CSVで一括インポートしたい。
```

---

## 機能要件

### 必須（Must）

- [x] ユーザー一覧を取得できる（管理者のみ）
- [x] ユーザー詳細を確認できる（管理者のみ）
- [x] 新規ユーザーを作成できる（管理者のみ）。作成時に初期設定メールを自動送信
- [x] まとめて登録（複数行フォーム入力）ができる
- [x] CSV取り込みによる一括登録ができる（name,email,role,status 形式）
- [x] ユーザーのロール（admin/member）とステータス（active/inactive）を変更できる（管理者のみ）

### 対象外（Won't）

- ユーザーの物理削除（`inactive` ステータスへの変更で対応）
- 管理者が他ユーザーのパスワードを直接変更する（パスワードリセット機能を利用）

---

## 受け入れ条件

- [x] 管理者でないユーザーがアクセスすると403エラー
- [x] 新規ユーザー作成時、初期設定用トークン（有効期限7日）のメールが送られる
- [x] 重複するメールアドレスで作成しようとすると409エラー
- [x] 更新できるのはロール・ステータスのみ（名前・メールは変更不可）

---

## 画面・遷移の概要

- 一覧画面（`/users`）: ロール・ステータスによるフィルタリング、キーワード検索
- 詳細画面（`/users/:id`）: ユーザー情報確認とロール・ステータス変更
- 新規作成画面（`/users/new`）: 1件フォーム入力 + CSV一括作成
