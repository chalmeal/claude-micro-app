# 画面設計書: ログイン画面

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |
| パス | `/login` |
| 認証 | 不要 |

---

## 概要

メールアドレスとパスワードでログインする画面。認証成功後はダッシュボードへ遷移する。

---

## 画面構成（ワイヤーフレーム）

```
+----------------------------------------+
|          [アプリ名・ロゴ]               |
|                                        |
|   メールアドレス                       |
|   [______________________________]     |
|                                        |
|   パスワード                           |
|   [______________________________]     |
|                                        |
|   [エラーメッセージ（ある場合）]       |
|                                        |
|         [ログイン]                     |
|                                        |
|   パスワードをお忘れの方はこちら       |
+----------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| メールアドレス入力 | type="email", required |
| パスワード入力 | type="password", required |
| エラーメッセージ | 認証失敗時に表示 |
| ログインボタン | submit, ローディング中は無効化 |
| パスワード忘却リンク | `/forgot-password` へ遷移 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | フォームのみ |
| 送信中 | ボタンをローディング表示、無効化 |
| 認証失敗 | 「メールアドレスまたはパスワードが正しくありません」を表示 |
| inactive ユーザー | 同上（区別しない） |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| フォーム送信 | POST `/api/auth/login` → 成功時にJWTをlocalStorageへ保存し `/` へ遷移 |
| パスワード忘却リンククリック | `/forgot-password` へ遷移 |

---

## 画面遷移

```
ログイン画面（/login）
  ├─ ログイン成功    → ダッシュボード（/）
  └─ パスワード忘却  → パスワード忘却画面（/forgot-password）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| POST | `/api/auth/login` | 認証・JWT取得 |
