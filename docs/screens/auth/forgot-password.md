# 画面設計書: パスワード忘却画面

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |
| パス | `/forgot-password` |
| 認証 | 不要 |

---

## 概要

パスワードを忘れたユーザーがメールアドレスを入力し、パスワードリセット用のメールを受け取るための画面。

---

## 画面構成（ワイヤーフレーム）

```
+----------------------------------------+
|  ← ログインに戻る                       |
|                                        |
|   パスワードをお忘れの方               |
|   （説明文）                           |
|                                        |
|   メールアドレス                       |
|   [______________________________]     |
|                                        |
|       [リセットメールを送信]           |
|                                        |
|   ── 送信後メッセージ（あれば）──     |
+----------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| メールアドレス入力 | type="email", required |
| 送信ボタン | submit, 送信中は無効化 |
| 完了メッセージ | 送信後に表示（成功・失敗に関わらず同一文言） |
| ログインへ戻るリンク | `/login` へ遷移 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | フォームのみ |
| 送信中 | ボタンをローディング表示・無効化 |
| 送信後 | 「入力されたメールアドレスにリセット用リンクをお送りしました」を表示（アドレスが存在しない場合も同一表示） |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| フォーム送信 | POST `/api/auth/reset-password/request` → 常に成功メッセージを表示 |
| ログインへ戻るリンク | `/login` へ遷移 |

---

## 画面遷移

```
パスワード忘却画面（/forgot-password）
  └─ ログインへ戻る → ログイン画面（/login）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| POST | `/api/auth/reset-password/request` | リセットメール送信 |
