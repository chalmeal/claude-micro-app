# 画面設計書: パスワードリセット画面

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |
| パス | `/reset-password?token=<token>` |
| 認証 | 不要 |

---

## 概要

パスワードリセットメールのリンクから遷移する画面。URLのクエリパラメータに含まれるトークンを使い、新しいパスワードを設定する。

---

## 画面構成（ワイヤーフレーム）

```
+----------------------------------------+
|   パスワードのリセット                  |
|                                        |
|   新しいパスワード（8文字以上）        |
|   [______________________________]     |
|                                        |
|   [エラーメッセージ（ある場合）]       |
|                                        |
|       [パスワードを設定する]           |
+----------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| 新パスワード入力 | type="password", required, minlength=8 |
| エラーメッセージ | トークン無効・期限切れ等のエラー表示 |
| 設定ボタン | submit, 送信中は無効化 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | フォームのみ |
| 送信中 | ボタンをローディング表示・無効化 |
| トークン無効/期限切れ | エラーメッセージを表示（再度パスワード忘却画面からやり直しを促す） |
| 成功 | ログイン画面へ遷移 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| フォーム送信 | URLのtokenとnewPasswordをPOST → 成功時はログイン画面へ遷移 |

---

## 画面遷移

```
パスワードリセット画面（/reset-password?token=...）
  ├─ 設定成功  → ログイン画面（/login）
  └─ トークン無効 → エラー表示（同一画面）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| POST | `/api/auth/reset-password/confirm` | 新パスワード確定 |
