## 概要

既存の成績レコードのスコアを修正する画面。既存データをフォームの初期値として表示し、保存すると評価が再計算される。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------+
| ← 詳細に戻る                              |
|                                          |
| 成績編集                                  |
|                                          |
| 学生ID *                                 |
| [S001________________________] (編集不可) |
|                                          |
| 学生名 *                                 |
| [田中花子____________________]           |
|                                          |
| 科目 *                                   |
| [数学________________________] (編集不可) |
|                                          |
| スコア（0〜100） *                       |
| [85_____]                                |
|                                          |
| 年度 *                                   |
| [2026___] (編集不可)                     |
|                                          |
| 学期 *                                   |
| (●) 春学期  ( ) 秋学期 (編集不可)       |
|                                          |
| [エラーメッセージ（ある場合）]           |
|                                          |
|              [保存する]                  |
+------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 | 編集可否 |
| --- | --- | --- |
| 学生ID表示 | テキスト（読み取り専用） | 不可 |
| 学生名入力 | テキスト | 可 |
| 科目表示 | テキスト（読み取り専用） | 不可 |
| スコア入力 | 数値、0〜100 | 可 |
| 年度表示 | 数値（読み取り専用） | 不可 |
| 学期表示 | ラジオ（読み取り専用） | 不可 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 読み込み中 | スピナー |
| 表示中 | 既存データが入力済みのフォーム |
| 保存中 | ボタンをローディング表示・無効化 |
| 保存成功 | スナックバー「更新しました」→ 詳細ページへ遷移 |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| フォーム送信 | 確認ダイアログ表示 → PUT `/api/grades/:id` → 成功時に詳細ページへ遷移 |
| 詳細へ戻るリンク | `/grades/:id` へ遷移 |

---

## 画面遷移

```
成績編集（/grades/:id/edit）
  ├─ 保存成功      → 成績詳細（/grades/:id）
  └─ 詳細へ戻る   → 成績詳細（/grades/:id）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/grades/:id` | 既存データ取得（フォーム初期値） |
| PUT | `/api/grades/:id` | 成績更新 |
