## 概要

登録された成績を一覧表示する画面。評価・年度・学期・キーワードでフィルタリングでき、30件ずつのページネーションに対応。検索状態はsessionStorageに保持される。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------------------+
| 成績管理                             [+ 新規登録]    |
|                                                      |
| キーワード: [________]  評価: [全て▼]  年度: [____]  学期: [全て▼] |
| [検索]  [リセット]                                   |
|                                                      |
| +-------+--------+---------+-------+------+------+  |
| | 学生ID | 学生名  | 科目    | スコア | 評価 | 操作  |  |
| +-------+--------+---------+-------+------+------+  |
| | S001  | 田中   | 数学    |  85   |  A   | [詳細]|  |
| | S002  | 山田   | 英語    |  72   |  B   | [詳細]|  |
| +-------+--------+---------+-------+------+------+  |
|                                                      |
| [< 前へ]  1 / 5 ページ（全147件）  [次へ >]          |
+------------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| キーワード検索 | 学生名・学生ID・科目名の部分一致 |
| 評価フィルタ | S / A / B / C / D / F / 全て |
| 年度フィルタ | 数値入力 |
| 学期フィルタ | spring / fall / 全て |
| 成績テーブル | 学生ID・学生名・科目・スコア・評価・詳細リンク |
| ページネーション | 30件/ページ、前後ナビゲーション |
| 新規登録ボタン | `/grades/new` へ遷移 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 検索前（初期） | テーブルを表示（全件） |
| フィルタ後0件 | 「該当する成績がありません」を表示 |
| 読み込み中 | スケルトンローダー |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 検索フォーム送信 | GET `/api/grades` でデータ取得・一覧更新 |
| リセット | フィルタを初期化してAPI再取得 |
| ページ変更 | 対象ページのデータを取得 |
| 詳細リンク | `/grades/:id` へ遷移 |
| 新規登録ボタン | `/grades/new` へ遷移 |

---

## 画面遷移

```
成績一覧（/grades）
  ├─ 詳細リンク    → 成績詳細（/grades/:id）
  └─ 新規登録ボタン → 成績登録（/grades/new）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/grades` | 成績一覧取得（フィルタ・ページネーション） |
