---

## 概要

特定の成績レコードを表示する画面。編集・削除操作へのナビゲーションを提供する。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------+
| ← 成績一覧に戻る                          |
|                                          |
| 成績詳細                                  |
|                                          |
| 学生ID:    S001                          |
| 学生名:    田中花子                       |
| 科目:      数学                          |
| スコア:    85                            |
| 評価:      A                             |
| 年度:      2026                          |
| 学期:      春学期                        |
|                                          |
|        [編集]  [削除]                    |
|                                          |
| ── 削除確認ダイアログ（削除押下時）──    |
+------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| 成績情報表示 | 全フィールドを読み取り専用で表示 |
| 編集ボタン | `/grades/:id/edit` へ遷移 |
| 削除ボタン | 確認ダイアログを表示 |
| 削除確認ダイアログ | 「削除しますか？この操作は元に戻せません」 |
| 一覧へ戻るリンク | `/grades` へ遷移 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 読み込み中 | スピナー |
| 表示中 | 成績情報と操作ボタン |
| 削除確認中 | 確認ダイアログを表示 |
| 削除中 | 削除ボタンをローディング表示・無効化 |
| 削除成功 | 一覧画面へ遷移 |
| 404 | 「データが見つかりません」を表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 編集ボタン | `/grades/:id/edit` へ遷移 |
| 削除ボタン | 確認ダイアログを表示 |
| ダイアログ「削除する」 | DELETE `/api/grades/:id` → 成功時に一覧へ遷移 |
| ダイアログ「キャンセル」 | ダイアログを閉じる |
| 一覧へ戻るリンク | `/grades` へ遷移 |

---

## 画面遷移

```
成績詳細（/grades/:id）
  ├─ 編集ボタン    → 成績編集（/grades/:id/edit）
  ├─ 削除成功      → 成績一覧（/grades）
  └─ 一覧へ戻る   → 成績一覧（/grades）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/grades/:id` | 成績情報取得 |
| DELETE | `/api/grades/:id` | 成績削除 |
