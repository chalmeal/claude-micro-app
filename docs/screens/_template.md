## 概要

この画面が何をする画面か、誰が使うかを2〜3文で説明する。

---

## 画面構成（ワイヤーフレーム）

テキストでレイアウトの概要を表現する。

```
+--------------------------------------------------+
| ヘッダー                              [ログアウト] |
+--------------------------------------------------+
| [タイトル]                          [+ 新規作成]  |
|                                                  |
| 検索: [________] [検索]                          |
|                                                  |
| +------+----------+--------+------------------+ |
| | ID   | 名前     | 日付   | 操作             | |
| +------+----------+--------+------------------+ |
| | 001  | サンプル | 01/01  | [編集] [削除]    | |
| +------+----------+--------+------------------+ |
|                                                  |
| [< 前へ]  1 / 5  [次へ >]                       |
+--------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 | 備考 |
| --- | --- | --- |
| 検索フォーム | キーワード検索 | Enterキーでも検索 |
| 一覧テーブル | データを表形式で表示 | ソート機能あり |
| ページネーション | 20件ずつ表示 | |
| 新規作成ボタン | 登録画面へ遷移 | 管理者のみ表示 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| データあり | 一覧テーブルを表示 |
| データなし | 「データがありません」メッセージを表示 |
| 読み込み中 | スピナーを表示 |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 検索ボタンクリック | APIを呼び出してリストを更新 |
| 行クリック | 詳細画面へ遷移（`/example/:id`） |
| 削除ボタンクリック | 確認ダイアログを表示 → 確定でAPI呼び出し → 一覧更新 |
| 新規作成ボタンクリック | 登録画面へ遷移（`/example/new`） |

---

## 画面遷移

```
一覧画面（/example）
  ├─ 新規作成ボタン → 登録画面（/example/new）
  ├─ 行クリック    → 詳細画面（/example/:id）
  └─ 編集ボタン    → 編集画面（/example/:id/edit）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/example` | 一覧取得 |
| DELETE | `/api/example/:id` | 削除 |

詳細は API設計書（`docs/api/xxxx.md`）を参照。
