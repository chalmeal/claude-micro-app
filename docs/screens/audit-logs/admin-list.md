# 画面設計書: 監査ログ一覧画面

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |

| 認証 | 管理者のみ |

---

## 概要

システム内の管理操作ログを一覧で確認する画面。ユーザー・アクション・日時でフィルタリングでき、30件ずつのページネーションに対応。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------------------+
| 監査ログ                                              |
|                                                      |
| アクション: [__________]  ユーザー: [__________]    |
| 開始日時: [__________]  終了日時: [__________]      |
| [検索]  [リセット]                                   |
|                                                      |
| +------------+-------+----------+------+----------+ |
| | 日時        | ユーザー | アクション | 対象  | 詳細  | |
| +------------+-------+----------+------+----------+ |
| | 05/13 10:00 | admin@ | user.create | user | {...} | |
| | 05/13  9:30 | admin@ | batch.rerun | batch| {...} | |
| +------------+-------+----------+------+----------+ |
|                                                      |
| [< 前へ]  1 / 17 ページ（全500件）  [次へ >]        |
+------------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| アクションフィルタ | アクション名のテキスト入力（例: `user.create`） |
| ユーザーIDフィルタ | ユーザーIDのテキスト入力 |
| 開始日時フィルタ | ISO 8601形式の日時入力 |
| 終了日時フィルタ | ISO 8601形式の日時入力 |
| 監査ログテーブル | 日時・ユーザーメール・アクション・対象種別・詳細（JSON展開可） |
| ページネーション | 30件/ページ |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| データあり | テーブルを表示 |
| データなし | 「該当するログがありません」を表示 |
| 読み込み中 | スケルトンローダー |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 検索フォーム送信 | GET `/api/audit-logs` でデータ取得・一覧更新 |
| リセット | フィルタを初期化してAPI再取得 |
| ページ変更 | 対象ページのデータを取得 |
| 詳細セル | JSONをフォーマットして表示（展開/折りたたみ） |

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/audit-logs` | 監査ログ一覧取得（フィルタ・ページネーション） |
