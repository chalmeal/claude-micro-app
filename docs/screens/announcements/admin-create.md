# 画面設計書: お知らせ新規作成画面

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |
| パス | `/admin/announcements/new` |
| 認証 | 管理者のみ |

---

## 概要

新しいお知らせを作成する管理者向け画面。タイトル・本文・日付・カテゴリを入力して登録する。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------+
| ← お知らせ管理に戻る                      |
|                                          |
| お知らせ新規作成                          |
|                                          |
| タイトル *                               |
| [______________________________]         |
|                                          |
| 本文 *                                   |
| [                              ]         |
| [                              ]         |
| [______________________________]         |
|                                          |
| 日付 *                                   |
| [YYYY-MM-DD__________]                   |
|                                          |
| カテゴリ *                               |
| ( ) 重要  (●) お知らせ  ( ) メンテナンス |
|                                          |
| [エラーメッセージ（ある場合）]           |
|                                          |
|              [作成する]                  |
+------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 | バリデーション |
| --- | --- | --- |
| タイトル入力 | テキスト | 必須 |
| 本文入力 | テキストエリア | 必須 |
| 日付入力 | `YYYY-MM-DD` 形式 | 必須 |
| カテゴリ選択 | ラジオボタン | 必須（important / info / maintenance） |
| 作成ボタン | submit | — |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | 空フォーム |
| 送信中 | ボタンをローディング表示・無効化 |
| 作成成功 | 管理一覧へ遷移 |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| フォーム送信 | POST `/api/announcements` → 成功時に管理一覧へ遷移 |
| 管理一覧へ戻るリンク | `/admin/announcements` へ遷移 |

---

## 画面遷移

```
お知らせ新規作成（/admin/announcements/new）
  ├─ 作成成功      → お知らせ管理一覧（/admin/announcements）
  └─ 一覧へ戻る   → お知らせ管理一覧（/admin/announcements）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| POST | `/api/announcements` | お知らせ作成 |
