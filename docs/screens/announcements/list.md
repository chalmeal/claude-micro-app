## 概要

認証済みユーザーがシステムのお知らせを閲覧する画面。30件ずつのページネーションで表示する。編集・削除操作はできない（管理者向け画面から行う）。

---

## 画面構成（ワイヤーフレーム）

```
+--------------------------------------------------+
| お知らせ                                          |
|                                                  |
| +----------+-----------------------------+-----+ |
| | カテゴリ  | タイトル                    | 日付 | |
| +----------+-----------------------------+-----+ |
| | 重要      | システムメンテナンスのお知らせ | 05/15 | |
| | お知らせ  | 新機能リリースのご案内      | 05/10 | |
| +----------+-----------------------------+-----+ |
|                                                  |
| [< 前へ]  1 / 3 ページ  [次へ >]                |
+--------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| お知らせテーブル | カテゴリバッジ・タイトル・日付を表示 |
| ページネーション | 30件/ページ |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| データあり | テーブルを表示 |
| データなし | 「お知らせはありません」を表示 |
| 読み込み中 | スケルトンローダー |
| エラー | エラーメッセージを表示 |

---

## カテゴリのラベル表示

| 値 | 表示ラベル |
| --- | --- |
| `important` | 重要 |
| `info` | お知らせ |
| `maintenance` | メンテナンス |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| ページ変更 | 対象ページのデータを取得・表示 |

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/announcements` | お知らせ一覧取得 |
