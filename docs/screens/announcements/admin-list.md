# 画面設計書: お知らせ管理一覧画面（管理者向け）

| 項目 | 内容 |
| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |
| ステータス | 完了 |
| パス | `/admin/announcements` |
| 認証 | 管理者のみ |

---

## 概要

管理者がお知らせを一覧で管理する画面。新規作成・編集・削除操作ができる。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------------------+
| お知らせ管理                         [+ 新規作成]    |
|                                                      |
| +----------+-----------------------------+-----+----+-----+ |
| | カテゴリ  | タイトル                    | 日付 | 操作      | |
| +----------+-----------------------------+-----+----+-----+ |
| | 重要      | システムメンテナンスのお知らせ | 05/15 | [編集][削除] | |
| | お知らせ  | 新機能リリースのご案内      | 05/10 | [編集][削除] | |
| +----------+-----------------------------+-----+----+-----+ |
+------------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| お知らせテーブル | カテゴリ・タイトル・日付・編集/削除ボタン |
| 新規作成ボタン | `/admin/announcements/new` へ遷移 |
| 編集ボタン | `/admin/announcements/:id/edit` へ遷移 |
| 削除ボタン | 確認ダイアログを表示してから削除 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| データあり | テーブルを表示 |
| データなし | 「お知らせはありません」を表示 |
| 読み込み中 | スケルトンローダー |
| 削除確認中 | 確認ダイアログを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 新規作成ボタン | `/admin/announcements/new` へ遷移 |
| 編集ボタン | `/admin/announcements/:id/edit` へ遷移 |
| 削除ボタン | 確認ダイアログ表示 → DELETE `/api/announcements/:id` → 一覧更新 |

---

## 画面遷移

```
お知らせ管理一覧（/admin/announcements）
  ├─ 新規作成ボタン → 新規作成画面（/admin/announcements/new）
  └─ 編集ボタン    → 編集画面（/admin/announcements/:id/edit）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/announcements` | 一覧取得 |
| DELETE | `/api/announcements/:id` | 削除 |
