## 概要

バッチジョブの一覧・スケジュール編集・有効/無効切り替え・手動再実行・実行履歴・実行ログを1画面で管理する。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------------------+
| バッチジョブ管理                                      |
|                                                      |
| +----------+--------+-----------+----------+-------+ |
| | バッチ名  | 状態   | 最終実行   | 次回実行  | 操作  | |
| +----------+--------+-----------+----------+-------+ |
| | 日次集計  | ✓成功  | 05/13 0:00| 05/14 0:00| [▼] | |
| | 週次レポート| ✗失敗 | 05/11 9:00| 05/18 9:00| [▼] | |
| +----------+--------+-----------+----------+-------+ |
|                                                      |
| ── バッチ展開時（アコーディオン） ───               |
| スケジュール:  [毎日▼]  [00:00]  [保存]             |
| 有効:         [ON/OFF トグル]                        |
| 再実行:       [今すぐ実行]                           |
|                                                      |
| 実行履歴:                                            |
| +----------+--------+-----------+                   |
| | 開始時刻  | 状態   | 所要時間  |                   |
| +----------+--------+-----------+                   |
| | 05/13 0:00| 成功   | 1.2s      | [ログを見る]      |

| [さらに読み込む]                                     |

+------------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| バッチ一覧テーブル | 名前・ステータス・最終実行・次回実行 |
| スケジュール編集フォーム | 頻度（分毎/時毎/日毎/週毎/月毎）・時刻・曜日/日の設定 |
| 有効/無効トグル | ON/OFFで即時PATCH |
| 今すぐ実行ボタン | POST でバッチを即時トリガー |
| 実行履歴テーブル | 開始時刻・ステータス・所要時間 |
| ログ表示 | タイムスタンプ・ログレベル・メッセージ（info/warn/error 色分け） |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 一覧読み込み中 | スケルトンローダー |
| バッチ展開 | スケジュール・トグル・再実行ボタン・実行履歴を表示 |
| スケジュール保存中 | 保存ボタンをローディング表示 |
| 実行中 | ステータスに「実行中」バッジを表示 |
| 実行成功 | ステータスに「成功」バッジを表示 |
| 実行失敗 | ステータスに「失敗」バッジを赤色表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| バッチ行クリック | アコーディオン展開 |
| スケジュール保存 | PUT `/api/batches/:id/schedule` → 成功時に `nextRunAt` を更新表示 |
| トグル切り替え | PATCH `/api/batches/:id/enabled` → 即時反映 |
| 今すぐ実行 | POST `/api/batches/:id/rerun` → スナックバー表示 |
| 実行履歴「さらに読み込む」 | GET `/api/batches/:id/runs?offset=N` で追加読み込み |
| ログを見る | GET `/api/batches/runs/:runId/logs` → ログを展開表示 |

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/batches` | バッチ一覧取得 |
| GET | `/api/batches/:id/runs` | 実行履歴取得 |
| GET | `/api/batches/runs/:runId/logs` | 実行ログ取得 |
| PUT | `/api/batches/:id/schedule` | スケジュール更新 |
| PATCH | `/api/batches/:id/enabled` | 有効/無効切り替え |
| POST | `/api/batches/:id/rerun` | 即時再実行 |
