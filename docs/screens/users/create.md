---

## 概要

新規ユーザーを登録する管理者向け画面。1件登録・まとめて登録・CSV取り込みの3モードを持つ。登録完了時に対象ユーザーへ初期パスワード設定メール（有効期限7日）が自動送信される。

---

## 画面構成（ワイヤーフレーム）

```
+------------------------------------------+
| ← ユーザー一覧に戻る                      |
|                                          |
| ユーザー新規作成                          |
|                                          |
| [1件登録] [まとめて登録] [CSV取り込み]   |
| ─────────────────────────                |
|                                          |
| ── 1件登録 ──                           |
| 名前 *                                   |
| [______________________________]         |
|                                          |
| メールアドレス *                         |
| [______________________________]         |
|                                          |
| ロール: ( ) admin  (●) member            |
| ステータス: (●) active  ( ) inactive     |
|                                          |
|              [登録する]                  |
|                                          |
| ── まとめて登録 ──                      |
| +------+----------+--------+----------+ |
| | 名前 | メール   | ロール | ステータス | |
| +------+----------+--------+----------+ |
| | [__] | [______] | [▼]   | [▼]      | |
| | [__] | [______] | [▼]   | [▼]      | |
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| タブ切り替え | 1件登録 / まとめて登録 / CSV取り込みの3モード |
| 1件登録フォーム | name・email・role・status 入力 |
| まとめて登録フォーム | 複数行テーブル。行追加/削除が可能（初期3行）。空の行は無視 |
| CSV取り込みフォーム | CSVファイル選択・解析後プレビュー表示・エラー行の案内 |
| 確認ダイアログ | 登録内容を確認してから実行 |

---

## 1件登録フォームのフィールド

| フィールド | 型 | 必須 | バリデーション | 初期値 |
| --- | --- | --- | --- | --- |
| name | string | 必須 | 空白でない | '' |
| email | string | 必須 | 空白でない | '' |
| role | 'admin' \| 'member' | 必須 | — | 'member' |
| status | 'active' \| 'inactive' | 必須 | — | 'active' |

---

## まとめて登録フォームのフィールド（1行あたり）

| フィールド | 型 | 説明 |
| --- | --- | --- |
| name | string | 空白の行は送信時に除外 |
| email | string | 同上 |
| role | 'admin' \| 'member' | セレクトボックス |
| status | 'active' \| 'inactive' | セレクトボックス |

---

## CSV取り込みフォーマット

```csv
name,email,role,status
田中太郎,tanaka@example.com,member,active
鈴木花子,suzuki@example.com,admin,active
```

| カラム | 必須 | 許容値 |
| --- | --- | --- |
| name | 必須 | 任意の文字列 |
| email | 必須 | 任意の文字列 |
| role | 必須 | `admin` または `member` |
| status | 必須 | `active` または `inactive` |

バリデーションエラーの行はスキップされ、有効な行のみ登録される。

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| 初期表示 | 1件登録フォーム（タブ選択中） |
| CSV解析後 | プレビューテーブルとエラーメッセージ（エラーあれば）を表示 |
| 確認ダイアログ | 登録内容のサマリーを表示 |
| 送信中 | ボタンをローディング表示・無効化 |
| 作成成功 | ユーザー一覧へ遷移 |
| メール重複エラー | `N行目: このメールアドレスは既に使用されています` を表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 1件登録フォーム送信 | 確認ダイアログ → POST `/api/users` → 一覧へ遷移 |
| まとめて登録フォーム送信 | 確認ダイアログ → POST `/api/users` を有効行ごとに順次実行 → 一覧へ遷移 |
| CSVアップロード | フロントエンドでCSVをパース → プレビュー表示 |
| CSV登録ボタン | 確認ダイアログ → POST `/api/users` を各行に対して順次実行 → 一覧へ遷移 |
| 一覧へ戻るリンク | `/users` へ遷移 |

---

## 画面遷移

```
ユーザー新規作成（/users/new）
  ├─ 登録成功      → ユーザー一覧（/users）
  └─ 一覧へ戻る   → ユーザー一覧（/users）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| POST | `/api/users` | ユーザー新規作成（モードに関わらず1件ずつ呼び出し） |

---

## 備考・決定事項

- まとめて登録・CSV取り込みともに、専用の一括登録APIはなく POST `/api/users` を件数分順次呼び出す
- 途中でエラーが発生した場合、エラー行より前の登録は完了済みとなる（ロールバックなし）
