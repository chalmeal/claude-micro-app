# 画面設計書: ユーザー一覧画面

| --- | --- |
| 作成日 | 2026-05-13 |
| 更新日 | 2026-05-13 |

| パス | `/users` |
| 認証 | 管理者のみ |

---

## 概要

登録されている全ユーザーを一覧表示する管理者向け画面。ロール・ステータスでフィルタリングしたり、名前・メールで検索できる。

---

## 画面構成（ワイヤーフレーム）

```
+--------------------------------------------------+
| ユーザー管理                      [+ 新規作成]   |
|                                                  |
| 検索: [__________]  ロール: [全て▼]  ステータス: [全て▼]  [検索] [リセット] |
|                                                  |
| +--------+------------------+--------+---------+--------+ |
| | 名前   | メール           | ロール | ステータス | 操作  | |
| +--------+------------------+--------+---------+--------+ |
| | 山田   | yamada@...       | admin  | active  | [詳細] | |
| | 田中   | tanaka@...       | member | active  | [詳細] | |
| +--------+------------------+--------+---------+--------+ |
+--------------------------------------------------+
```

---

## コンポーネント一覧

| コンポーネント | 説明 |
| --- | --- |
| 検索フォーム | 名前・メールのキーワード検索 |
| ロールフィルタ | 全て / admin / member |
| ステータスフィルタ | 全て / active / inactive |
| ユーザーテーブル | 名前・メール・ロール・ステータス・詳細リンク |
| 新規作成ボタン | `/users/new` へ遷移 |

---

## 状態と表示

| 状態 | 表示内容 |
| --- | --- |
| データあり | テーブルを表示 |
| データなし（フィルタ後） | 「該当するユーザーがいません」を表示 |
| 読み込み中 | スピナーまたはスケルトン |
| エラー | エラーメッセージを表示 |

---

## ユーザー操作と挙動

| 操作 | 挙動 |
| --- | --- |
| 検索・フィルタ変更 | フロントエンド側でリストをフィルタリング（API再呼び出しなし） |
| リセットボタン | フィルタ・検索をクリア |
| 詳細リンク | `/users/:id` へ遷移 |
| 新規作成ボタン | `/users/new` へ遷移 |

---

## 画面遷移

```
ユーザー一覧（/users）
  ├─ 詳細リンク    → ユーザー詳細（/users/:id）
  └─ 新規作成ボタン → ユーザー新規作成（/users/new）
```

---

## 使用するAPI

| メソッド | エンドポイント | 用途 |
| --- | --- | --- |
| GET | `/api/users` | ユーザー一覧取得（初回ロード時） |
